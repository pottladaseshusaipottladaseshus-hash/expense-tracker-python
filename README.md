# Expense Tracker (Python)

## About the Project

This is a simple personal expense tracker built using Python. The application allows users to record and analyze their daily expenses. It automatically creates and manages a data file to store expense records.

## Features

* Add daily expenses (date, category, amount)
* View all expenses
* Analyze total and category-wise spending
* Visualize data using graphs
* Automatically creates a CSV file for storage

## Technologies Used

* Python
* Pandas
* Matplotlib

## How It Works

The program stores expense data in a CSV file that is automatically created when the application runs for the first time. It uses Pandas for analysis and Matplotlib for visualization.

## How to Run

1. Install libraries:
   pip install pandas matplotlib

2. Run:
   python main.py

---

This project demonstrates basic data analysis and file handling using Python in a practical way.
